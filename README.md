## 📱 Funcionalidades

### Navegação
- Smooth scroll para seções
- Nav bar com efeito no scroll
- Links ativos destacados

### Modais
- Clique em qualquer categoria de trabalho
- Modal full-screen com conteúdo detalhado
- Fecha com botão X, ESC ou clique fora
- Scroll bloqueado quando modal aberto

### Animações
- Fade in de elementos no scroll

- Contadores animados nas estatísticas
- Hover effects em cards
- Transições suaves entre estados

## 🔧 Customização

### Trocar Cores
Edite as variáveis CSS em `style.css`:
```css
:root {
    --color-accent: #DC2626; /* Sua cor aqui */
}
```

### Adicionar/Editar Projetos
Os projetos estão em modais no `index.html`. Busque por:
```html
<div id="modal-categoria" class="modal">
```

### Ajustar Foto
A foto está em:
```html
<img src="URL_DA_FOTO" alt="Leonardo Barragana">
```
